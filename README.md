# Covid-Tracker
This is a website which gives the stats of Covid across the globe.

This is built with Reactjs, Maetrial-UI and an Covid-19 API

To host this website in your local host,Follow these steps:

#) Clone the repo
#) cd [repo_name]
#) npm install
#) npm start
